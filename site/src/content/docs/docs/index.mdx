import { Card, CardGrid, LinkButton } from '@astrojs/starlight/components';

Penates is a small self-hosted server that runs on your Mac or Linux box. It shows every running coding-CLI session in one dashboard and lets you drop into the terminal from any browser — your Mac, iPad, or phone, on your network or over the internet. No cloud relay: your machine, your sessions.

You start sessions with **Claude Code** (Anthropic), **Codex** (OpenAI), or **Antigravity** (Google `agy`) from a CLI picker, each with tiered approval/sandbox variants. Claude Code is the most deeply integrated (hook-based notifications, usage tracking, image paste); the others run as full terminal sessions with their own login.
