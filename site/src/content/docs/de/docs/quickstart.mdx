<Steps>

1. **Voraussetzungen installieren:** Node ≥20, tmux und mindestens Claude Code. Die vollständige Liste mit Einzeilern findest du unter [Voraussetzungen](/de/docs/install/requirements).

2. **Klonen und Setup ausführen**

   ```bash
   cd ~
   git clone https://github.com/DerRemo/penates.git
   cd penates
   chmod +x setup.sh
   ./setup.sh
   ```

   `setup.sh` installiert die Abhängigkeiten, schreibt eine `.env` mit einem zufälligen Auth-Token, richtet den Auto-Start ein und startet den Server.

3. **Den Hub öffnen** unter `http://localhost:3333`. Beim ersten Besuch fragt der Browser nach dem Token. Das ist der Wert von `AUTH_TOKEN` in `penates/.env` (jederzeit nachschlagen mit `grep AUTH_TOKEN penates/.env`).

4. **Eine Session starten:** CLI im Dashboard auswählen, fertig.

</Steps>

<Aside title="Ein-Zeilen-Install">
Lieber einen geführten Installer? `curl -fsSL --proto '=https' --tlsv1.2 https://raw.githubusercontent.com/DerRemo/penates/main/install.sh | bash` führt durch alles. Siehe [Installation](/de/docs/install/installation).
</Aside>
